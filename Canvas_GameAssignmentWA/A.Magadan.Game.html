The assignment was pretty easy as most of the elements for the game were kept in a div container which allowed me to just move and resize the container and make the elements fill it. Changing the game colors was also easy as all it took was to search for the parts in the code that drew the objects and change their fill styles. However I struggled with trying to zoom in the game.
